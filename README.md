# Marketing Campaign Analytics

This project analyzes marketing campaign performance using Python.

Technologies used:
- Python
- Pandas
- Matplotlib
- Seaborn

Metrics analyzed:
- Click Through Rate
- Conversion Rate
- ROI

Insights:
- Identify best performing channels
- Understand customer segments
